# Final Pre-Submission Checklist

Before you submit anything to ai-bounties.hedera.com, confirm all items:

- [ ] Repo is public on GitHub with good name and description
- [ ] At least some commits show activity during the campaign window
- [ ] Live demo is deployed and publicly accessible (test it yourself)
- [ ] Feedback GitHub Issue has been opened on the official kit repo and link is ready
- [ ] 4-6 high-quality images/screenshots/GIFs prepared (UI, policy blocking, success flow, HCS audit, etc.)
- [ ] Payout wallet address ready
- [ ] You have re-read the official terms at least once
- [ ] SUBMISSION_FORM_TEXT.txt has been updated with your actual repo URL, demo URL, feedback URL, and wallet
- [ ] You have explicitly confirmed with Grok Build that we are cleared to proceed with the actual submission step

Only after all the above + explicit confirmation should you go to the bounty site and submit.

This package is designed so that once you have the URLs, filling the form takes < 3 minutes.
